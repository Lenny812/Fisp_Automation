# FISP Report Generator

A web-based tool for generating NYC DOB-compliant Facade Inspection Safety Program (FISP/Local Law 11) reports.

## Features
- Complete building information form
- QEWI and inspector details
- Safe/SWARMP/Unsafe condition tracking with photos
- All DOB-required appurtenances
- 3 professional report formats (Classic, Modern, Detailed)
- Photo uploads for conditions and elevations
- Generates downloadable Word documents
